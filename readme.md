# sfcurve: 2x2 anc 3x3 Space-Filling Curves


## Install


```r
devtools::install_github("jokergoo/sfcurve")
```

## Usage

Hilbert curve (2x2)

```r
p = sfc_hilbert("I", "111")
```

Peano curve (3x3)

```r
p = sfc_peano("I", "111")
```

Meander curve (3x3)

```r
p = sfc_meander("I", "111")
```
